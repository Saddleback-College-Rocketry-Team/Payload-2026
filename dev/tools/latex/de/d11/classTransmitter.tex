\subsubsection*{template$<$typename SerialT$>$\newline
class Transmitter$<$ Serial\+T $>$}
Template class for a transmitter device using serial communication. 

This class initializes and configures a transmitter module using AT commands. It allows setting the network ID and device address over a serial interface.
