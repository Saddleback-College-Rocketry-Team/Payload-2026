\subsubsection*{template$<$typename SDT$>$\newline
class SDManager$<$ SDT $>$}
Template class for SD card operations. 

This class provides helper functions for interacting with an SD card. It allows checking if a file exists, creating a CSV file, and writing headers if the file does not already exist.
